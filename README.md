# METAR Weather Tray

Приложение для Windows 10, отображающее погодные данные METAR в системном трее.

## Описание

Приложение получает, декодирует и отображает данные METAR (Meteorological Aerodrome Report) в системном трее Windows с иконками облачности и осадков, а также сохраняет историю изменений.

## Установка

```bash
go build -o metar-tray.exe ./cmd/metar-tray
```

## Использование

После запуска приложение появится в системном трее Windows.
